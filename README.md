# 🎬 Movie Review Platform

## Описание проекта
Веб-платформа для просмотра информации о фильмах: рейтинги, кассовые сборы, 
обзоры критиков и пользователей. Пользователи могут искать фильмы, читать 
и оставлять рецензии, оценивать фильмы.

## Функционал
- Просмотр карточек фильмов с детальной информацией
- Кассовые сборы и рейтинги
- Обзоры критиков и пользовательские рецензии
- Регистрация/авторизация пользователей
- Возможность оставлять свои отзывы и оценки

## Технологии
- Backend: Django, Django REST Framework
- Frontend: Django Templates, CSS
- База данных: SQLite
